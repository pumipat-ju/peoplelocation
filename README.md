# AI CCTV Monitoring System
ระบบตรวจจับคนและ Tracking ด้วย YOLOv8 + FastAPI

## วิธีติดตั้ง
1. Clone โปรเจกต์นี้ไป
2. สร้าง venv: `python -m venv venv`
3. ติดตั้ง Library: `pip install -r requirements.txt`
4. รันโปรเจกต์: `python main.py`

## การใช้งาน
- เข้าใช้งานผ่านเบราว์เซอร์ที่พอร์ต 8000
- สามารถเพิ่มกล้องผ่านหน้าเว็บได้ (Webcam ใส่เลข 0)
